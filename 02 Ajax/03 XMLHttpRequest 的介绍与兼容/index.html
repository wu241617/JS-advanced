XMLHttpRequest 类

Ajax 技术的核心是XMLHttpRequest类，简称 XHR，它允许脚本异步调用 HTTP API。 浏览器在XMLHttpRequest类上定义了 HTTP API，这个类的每个实例都表示一个 独立 的 请求/响应 对象， 并且这个实例对象上的属性和方法允许 指定细节 和 提取响应数据。 当然，既然XMLHttpRequest是个"类"，那么在使用时，必须做的第一件事就是实例化它： // 变量xhr来表示这个实例:var xhr = new XMLHttpRequest();

XMLHttpRequest 的兼容

作为现有标准的新方法，绝大多数的浏览器都支持XMLHttpRequest。 但在 IE7 之前的版本（IE5、IE6）并不支持XMLHttpRequest()构造函数，它们需要使用ActiveX对象进行模拟： var xhr = new ActiveXObject("Microsoft.XMLHTTP");

XMLHttpRequest的兼容写法可以是这样的：

var xhr = window.XMLHttpRequest ? new window.XMLHttpRequest() : new ActiveXObject("Microsoft.XMLHTTP");
